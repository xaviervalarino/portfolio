Designing a customizable dashboard

Description

GEARS was a SaaS application for monitoring devices on corporate networks. Network administrators would create security policies by configuring rules for device compliance. The application would then report back how many vdevices were out of compliance, tallying each broken rule as an issue. We had recently added new metrics for triaging devices and needed to update the dashboard to reflect this change. This gave us the opportunity to make the dashboard more useful to customers with various network topologies.

Goal

Provide administrators with tools to personalize their dashboards so that they could easily get the information that was important to them.

Role
